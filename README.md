# Skoolnet - Educational Management Platform

Skoolnet is a comprehensive educational management platform built with React and Vite, designed to streamline administrative tasks, manage student data, and provide powerful analytics for educational institutions.

## Features

### Manager Dashboard
- **Dashboard Overview**: Real-time metrics and system statistics
- **User Management**: Complete control over user accounts and permissions
- **Settings Configuration**: Centralized settings management with live preview
- **Notifications System**: Advanced notification management and campaign tools
- **Pricing Plans**: Flexible pricing plan management with custom plans

### Key Components

#### Settings Management
- **Comprehensive Settings**: Manage all site features from a central location
- **Categories**: User Management, Security, Communication, System Configuration, Features & Integrations, Content & Display
- **Live Preview**: Real-time preview of setting changes
- **Dynamic Controls**: Toggle switches, dropdown selects, and number inputs

#### Navigation System
- **SettingsNavigation Component**: Reusable navigation component with customizable props
- **Tab Functionality**: Support for tab-based navigation
- **Custom Styling**: Flexible styling options and responsive design

#### Pricing Management
- **Custom Plans**: Create and manage custom pricing plans
- **Manager Assignment**: Assign managers to specific plans
- **Indian Rupee Support**: Native support for Indian currency display
- **Interactive UI**: Dropdown menus, hover effects, and responsive design

#### Notifications
- **Campaign Management**: Create and manage notification campaigns
- **Template System**: Browse and use notification templates
- **Advanced Analytics**: Track notification performance and engagement

## Technology Stack

- **Frontend**: React 18 with functional components and hooks
- **Build Tool**: Vite for fast development and optimized builds
- **Styling**: Tailwind CSS for utility-first styling
- **Icons**: Custom AppIcon component with comprehensive icon library
- **State Management**: React useState for local state management

## Development Guidelines

### Component Structure
- Use functional components with hooks
- Follow React best practices
- Implement proper prop validation
- Use TypeScript for type safety (optional)

### Styling
- Use Tailwind CSS utility classes
- Follow consistent naming conventions
- Implement responsive design principles
- Use semantic HTML elements

### State Management
- Use useState for local component state
- Implement proper state updates
- Avoid direct state mutations
- Use useCallback for optimization
